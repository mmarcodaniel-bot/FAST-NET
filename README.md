## Ejemplo de código — Llamada a FASTY

Así se ve una consulta real al motor de IA desde el frontend en React:

```javascript
async function consultarFASTY(mensajeUsuario, contextoFinanciero) {
  const systemPrompt = `
    Eres FASTY, el asistente financiero de FastNet.
    Ingresos del mes: $${contextoFinanciero.ingresos}
    Gastos del mes: $${contextoFinanciero.gastos}
    Balance actual: $${contextoFinanciero.balance}
    Responde siempre en español, máximo 3 párrafos,
    de forma directa y accionable.
  `;

  const response = await fetch("https://api.anthropic.com/v1/messages", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      model: "claude-sonnet-4-20250514",
      max_tokens: 1000,
      system: systemPrompt,
      messages: [{ role: "user", content: mensajeUsuario }],
    }),
  });

  const data = await response.json();
  return data.content.map((c) => c.text || "").join("");
}
```

```javascript
// Ejemplo de categorización automática de una transacción
async function categorizarTransaccion(descripcion, monto) {
  const prompt = `
    Categoriza esta transacción financiera.
    Devuelve SOLO JSON: { "categoria": string, "tipo": "ingreso"|"gasto" }
    Transacción: "${descripcion}" por $${monto}
  `;
  const respuesta = await consultarFASTY(prompt, {});
  return JSON.parse(respuesta);
}
```
